# Dystopia世界觀基礎說明

這個世界原本並不是傳統意義上的奇幻世界，而是一個經歷過舊文明崩壞後，被魔法、幻想態、暗物質與空間異象重新塑造的世界。舊世界的科技體系在災變後失效，人類被迫在新的物理規則下重建文明，而後世所謂的「魔法」「靈魂」「闊神」「子空間」「虛空」等概念，都是在這個新世界中逐漸被理解、分類與制度化的結果。

Dystopia的世界處於「區域分治年代」。世界被分割成多個區域，每個區域都像是一個獨立社會，擁有自己的政治制度、文化、科技水準、危機型態與地理特徵。區域之間由「區間界」與通道門體系隔離，不同地區的人不一定能自由往來。這種制度表面上是為了防止文化、科技與軍事力量失衡，實際上也讓整個世界變成一個被管理、被限制、被觀測的結構。

這套秩序的建立與賽蒙密切相關。賽蒙是理想國的創立者，也是區間界、身分鎖與跨區管理制度的重要建立者。他並不是神，而是在時間流異象後崛起、試圖用制度壓制混亂的人。他讓世界從大規模衝突中安定下來，但他的問題在於，他把「穩定」誤認為「永恆」。

# 世界的底層規則

這個世界有三套重要底層系統：魔法、量子空間、精神／幻想態。

魔法並不是單純的神秘力量，而是一種以「魔力」為基礎的能量反應。魔力由魔素構成，魔素具有不同性質；一代時期主要可分為烈、固、流、虛四種。魔法需要介質與渠道，不能憑空自由塑形。大氣中存在魔素，生物體內也會生成魔力，兩者會互相平衡。魔力濃度過高時，可能造成生物侵染，使其變成異常存在。

更深層來說，魔力的根源是低純度暗物質。暗物質是不應存在於世界中的物質，會自然聚合、降解，並且擁有隔絕性與腐蝕性。高純度暗物質能讓空間與外界時間流不同步，也能吞噬空間中的物質並產生更多暗物質。這也是虛空危機的根本。

量子與空間規則則決定了子空間、異象與相依世界如何成立。這個世界中，某些空間或實體必須被觀測才能確定存在；未被觀測時，其內部時間與空間可能呈現靜止狀態。子空間需要媒介或基點才能穩定觀測，而相依世界則是存在於不同次元、與本世界不平行干涉的特殊世界。

精神學則解釋了靈魂、固有力、闊神與幻想態。生物由軀體、靈魂與意識構成；意志足夠強大時，靈魂能展現出影響現實的力量，稱為固有力。當靈魂進一步變革並與意識結合成「精神」後，固有力會進階成闊神。闊神能具現化概念或展開規則，但不能無限制常駐使用。

幻想態是這個世界背後最重要的概念生命。它們是外來物種，能以意識、信仰、契約等意念為食。現存幻想態大致分為基幻想態、寄生幻想態與游離幻想態。寄生幻想態寄宿於知性生命體內，成為靈魂；基幻想態則被信仰為神祇；游離幻想態最危險，容易擾亂魔力流並引發異象。

# 一代世界的社會結構

一代時期的世界是被分區管理的立方世界。每個區域都有明顯主題與社會困境，不只是地圖風格不同，而是整個文明邏輯都不同。

一區「無名之地」偏向荒蕪、遺民、廢墟與重建。這裡有枯木惡地、地神之痕、灰濛泥帶、研究站遺址、瓦礫廢墟與新生之城等地點，是適合安排流亡者、殘民、邊境聚落與早期探索型劇情的區域。

二區「寧謐墓地／艾倫狄爾」是地表墓園與地下地牢並存的區域。這裡有靈留之地、留魂井、地牢之心、冒險者社區與外界三個臨界點，整體氣質偏向靈魂、死亡、探險、公會與地下生存。

三區「灰寂工場／坎貝爾」是高度管制、勞動化與政府集中管理的工業區。中央政府、軍營、叛亂者監獄、採礦坑道、資源倉庫、舊會議據點等結構讓它很適合做成壓迫制度、礦工開局、叛逃、政變與地下組織劇情的舞台。

四區「晶簇空域／克爾納亞」是晶體、空中都市、魔學園區與政治派系並存的區域。稜晶之塔、魔塔、星空集映室、空中公園、靈獸之森、魔學園區與採晶場使它很適合安排學術、魔法研究、政治鬥爭與高處空域探索。

五區「熾熱岩床／瑪扈洛」是火山、岩漿、鍛爐、領主直轄與改革義軍衝突的地區。莫帝火山、無管轄區、間隙堡壘、熔火鍛爐、灼裂雙城、嘆息之城與暴雨異象後的新地，都能支撐災害、領地政治、礦物資源與革命線。

六區「綠野王國／瑟雷斯提亞」是人類王國、家族領地、魔物社會與邊境森林並存的地區。人類方有格林、尼爾福海姆、克雷布克、阿爾狄肯等家族領地與聖王朝都城；魔物方則有泥沼地帶、夜幕山脈、迷失密林、萬象之城與魔王城。這裡非常適合做種族衝突、王權、家族、魔物自治與邊境生存劇情。

七區「危難城市／瓦洛內斯」是企業、犯罪、階級與地下異常最集中的都市區。它分成煤焦、煉區、螢石、固鋼、青銅、黃銅、黑金、流銀、白金、黃金、鑽石、星辰等區塊，每個區塊都有對應企業與產業。表側是企業城市，裏側則有考克施工地、鍋爐迷層與嚴肅清理區，是整個世界最適合做黑市、企業戰爭、暗殺組織、人口販賣、地下怪物與高風險探索的地方。

八區「光彩迷都／盧米納」是賽博龐克、霓虹、機械、生體機械與高科技魔工學的區域。它的核心地形是輪盤，分成內層、中層與外層；內層有中央廣場與霓虹綜合區段，中層有機械平原與浮動平台，整體非常適合做高科技都市、AI、生體機械、資料潛流與魔法科技衝突。

理想國則是獨立於一般區域之外的特殊空間，是賽蒙建立的烏托邦式秩序核心。它曾經是世界制度的高位節點，也與一代主線後期的重大真相密切相關。

# 玩家所處的時代狀態

遊戲開始於「寬限期」，也就是虛空尚未明顯侵蝕世界之前。

這時候世界表面上仍然和平。各區域雖然有各自的矛盾，但還沒有全面惡化。戰爭區塊仍存在和平協議，企業倫理尚未完全崩壞，跨區制度仍然能維持基本秩序。玩家可以選擇以一般居民、冒險者、學者、傭兵、企業職員、逃亡者、魔物居民、貴族、管制局相關者、犯罪組織成員等身分開局。

但隨著時間推進，虛空會逐漸影響世界，使局勢分階段惡化。

第一階段「起源」是虛空初期影響世界。區域開始緊張，區間界進出被限制，只有獲得領主或管理者同意才能離開區域。各區域內部勢力開始備戰，理想國勢力顯現。這一階段適合讓玩家第一次感受到「原本和平的社會開始套上特殊規則」。

第二階段「進展」是人與人之間的爭執加劇，特殊異動開始出現。玩家是否創立、加入或依附某個派系會變得很重要。部分地圖解鎖，部分地圖封鎖。這一階段適合把遊戲從個人冒險推向勢力選擇。

第三階段「加劇」是重大災難與異象開始爆發。異象不是普通天災，而是現實規則被擾動後出現的極端災難，形式可以是暴雨、瘟疫、時間錯位、空間扭曲、靈魂異常、認知崩壞、暗物質外溢等。稱號使開始活躍，理想國人員也大量出現在立方世界中。

第四階段「終局」是虛空完全顯現。大氣暗物質濃度劇增，部分地區被侵蝕，暗物質實體開始出現。這些實體少見但極危險。人類社會開始內鬥，跨區秩序逐步崩壞，世界進入混亂狀態。

這條進程很適合做成遊戲內的「世界危機時鐘」。玩家不一定要立刻對抗虛空，但他們每次拖延、失敗、錯過事件或做出某些選擇，都可以讓危機時鐘往前推。

# 虛空是什麼

虛空不是單純的黑暗地點，也不是一般意義上的地獄或異世界。它是由暗物質侵蝕、量子隔絕、時空脫鉤與觀測失效共同孕生出的相依世界。

其起源可以追溯到舊世界災後。北極海深處的避難所被暗物質塗層包覆，原本目的是隔絕與保存，但暗物質的腐蝕性與隔絕性逐步使避難所失去與主世界的連結。它不是被毀滅，而是被切出世界之外，逐漸變成第一個真正意義上的相依世界。

虛空中的高層與居民逐漸被暗物質同化，成為最早的暗物質實體。米歇爾則是其中的例外：他在完全被侵蝕前與游離幻想態融合，使他的意識得以被承載，最後成為虛空之主。這也讓他不是單純的怪物，而是「由幻想態承載意識、反過來控制暗物質身體」的特殊存在。

在遊戲中，虛空可以被設計成一種「世界外側壓力」。它不一定一開始就作為敵方登場，而是透過現象滲透：

暗物質濃度上升、魔獸異常化、區域規則變得極端、交通受限、異象出現、夢境或觀測失真、特定人物開始聽見聲音、地圖邊界出現不該存在的道路。

虛空真正可怕的地方不是單純殺人，而是讓世界的成立方式開始失效。

# 能力與戰鬥系統的世界觀定位

這個世界的能力大致可以分成幾種來源。

第一是普通魔法。魔法需要魔力、介質、渠道與術式結構。它可以被學習、研究、製作成裝置，也可以透過刻印固定成可重複使用的功能。這使得魔法既是戰鬥手段，也是工程技術。

第二是固有力與闊神。這是個體意志與靈魂變革後展現出的特殊能力，通常專一且獨特。固有力偏向較小幅度影響現實，闊神則能展開更強的概念能力，但一定有條件、缺陷或副作用。

第三是擬魔。魔獸或怪物使用的魔法攻擊不等同普通魔法，而是其生物性或異常性造成的擬魔展現。魔獸、魔物、怪物、怪獸在世界觀中並非同義分類，牠們的來源、危險性與遭遇定位不同。

第四是虛空／暗物質相關能力。這類能力通常極危險，可能造成腐蝕、隔絕、觀測失效、空間錯位、精神侵蝕或存在異常。七區的巨獸雷穆流斯就是一種被暗物質包覆的高危怪物，能造成物理與虛空傷害，並透過暗物質包層抵擋致命傷。

# 怪物與遭遇風格

各區域的怪物不是隨機放置，而是與該地區環境、社會與異象高度相關。

一區偏向荒蕪與自然擬態，例如掃地泥潭會像流沙般吞噬獵物，木樹花會以甜氣引誘目標並吸血。

二區偏向靈魂、地牢與恐怖遭遇，例如古靈克洛瑞厄斯是騷靈與怨靈的巨大集合體，哭泣天使則透過逐步顯現、聲音警示與感官剝奪造成即死威脅。

五區偏向火山、岩漿與災害生態，例如熔岩巨像、陸山龜、熱焰鰻、熾鋒翼龍，以及暴雨異象後才出現的黴蛙與霧蜥蜴。

六區偏向森林、魔物社會與生態型魔獸，例如草原獵犬、瘴氣鱷、行軍蟻、抱根熊與魔音蝠；其中米可雷西斯的怪貓還帶有認知阻礙與虛空幻象。

七區則最適合放置都市異常與暗物質威脅，例如竊皮人、隱巢、偽街燈、耳扇鹿，以及雷穆流斯與無盡黑霧這類高階怪物遭遇。

對 TRPG 式文字遊戲來說，這些怪物不應只作為戰鬥單位，而應作為「場景機制」。例如哭泣天使是一個倒數型恐怖事件；隱巢是一個瘟疫調查事件；偽街燈是城市潛行事件；無盡黑霧則是心理耐受與空間逃脫事件。

# 主要勢力類型

這個遊戲可以讓玩家自由選擇立場，因此勢力設計很重要。

區域政府與領主體系代表官方秩序。它們不一定邪惡，但會優先維持區域穩定與自身權力。第一階段後，它們會限制跨區移動，成為玩家行動的重要阻礙。

理想國代表高位秩序與世界管理。它不只是普通政府，而是賽蒙建立的制度核心。當虛空危機加劇，理想國相關人員會開始顯現，對玩家而言可能是保護者、監視者、任務委託者或敵對勢力。

秩序管制局是理想國制度延伸出的管理／武力／研究機構，內部有萊妮絲、迪亞諾、雨果、卡娜、希爾等人物與派系。角色列表中明確列出了秩序管制局成員與其能力，能作為官方強權與特殊探員型 NPC 的來源。

夜咄是七區黑金區關聯的暗殺組織，包含德瓦那、薇爾妮特、艾雷文、麥克、德蕾莎、洛克、派翠西亞等人物。它適合在遊戲中作為黑市、暗殺、契約、情報與犯罪任務的核心勢力。

稱號使則是高危險性的自由強者。稱號使不是某種能力種族，而是在賽蒙時期前後出現、由自己或外人賦予稱號的強者。他們對各區政府來說很危險，因為一旦作亂就難以管控。賽蒙本人就曾被稱為「七色的魔法使」。角色列表中也列出了起源稱號使群體，包括賽蒙、貝里亞克、海德維格、梅雷迪絲、萊斯特、雷克斯與皮諾爾。

玩家自己的派系可以在第二階段後變得關鍵。這一點很適合做成遊戲核心：玩家不只是在跑任務，而是在末日前逐步建立一個能保護自己、控制資源、影響地區、對抗虛空或趁亂牟利的組織。

# 故事主題與玩家體驗

這個世界的核心主題不是單純的「對抗邪惡」，而是「在世界逐漸失序時，人如何選擇自己的位置」。

一代主線中，艾斯維爾的核心困境是從被「神」賦予使命開始，逐步發現自己其實並不理解真正想追求什麼；他曾以為自己在拯救世界，卻也讓局勢變得更難控制。

你的遊戲可以把這個主題轉成玩家體驗：

玩家不需要一開始就是救世主。
玩家可以只是礦工、學生、企業員工、冒險者、逃亡犯、魔物居民、醫療人員、傭兵、研究員或被困在某區域的普通人。

但隨著虛空侵蝕，所有人都會被迫面對選擇：

是逃離、保護、支配、研究、交易、揭露真相、建立派系，還是向虛空妥協？

這會比單純的主線任務更適合 LLM 作為 PM。因為 LLM 不只需要推劇情，而是要根據玩家身份、所在地區、當前階段、勢力關係與世界危機狀態，動態生成事件。
